## Usage
python main.py

## Controls
Shift
    D - Pencil
    L - Line
    C - Circle
    R - Rect
    T - Type

Ctrl
    ~ - Cycle palette
    F - Fill mode toggle
    E - Erase the canvas
    Q - Quit
